# Archon Renewed
A combat focused magic mod for Minecraft 1.20.1, available on the Fabric modloader.

This project is a fork of the original [Archon](https://github.com/Safrodev/Archon) mod created by Safro. The goal of the fork is fixing bugs and introducing some QOL improvements to the original mod.
